# LimonPOS User Guide

## Overview

LimonPOS is a Point of Sale (POS) app for restaurants and retail. This guide covers basic usage, screens, and input shortcuts.

---

## Getting Started

1. **Install** – Install the app from Google Play or an APK.
2. **Configure server** – On first use (or via Settings → Server URL), enter your backend server URL.
3. **Login** – Enter your 4-digit PIN to start.

---

## Login Screen

- **PIN entry** – Use the on-screen numpad or a connected keyboard.
- **Numpad keys**
  - `0–9` – Enter digits
  - `C` – Clear PIN
  - `⌫` (Backspace) – Delete last digit
  - **Enter** – Log in

---

## Main Screens

| Screen       | Description                                           |
|--------------|-------------------------------------------------------|
| **Floor Plan** | View and select tables; tap a table to open an order |
| **Order**    | Add products, modifiers, notes; view and edit cart   |
| **Payment**  | Enter amounts, select payment method, complete bill  |
| **Settings** | Printer setup, receipt size, server URL, logout      |
| **KDS**      | Kitchen Display for order management                 |

---

## Floor Plan

- Tap a **free** table to start a new order.
- Tap an **occupied** table to view or edit the current order.
- Use the **Home** icon to return to the floor plan from an order.

---

## Order Screen

- **Products** – Tap a product to add it to the cart.  
  If the product has modifiers, choose options first, then add.
- **Cart** – View items, change quantities, remove items.
- **Transfer table** – Move the order to another table (transfer icon).
- **Send to kitchen** – Send order to KDS.
- **Back** – Return to floor plan (Android back or back button).

---

## Payment Screen

- **Amount** – Enter amount received or adjust the total.
- **Payment method** – Choose cash, card, or other methods.
- **Split payment** – Add multiple payment methods as needed.
- **Complete** – Finalize the payment and close the table.

---

## Keyboard Shortcuts & Input

When using a **physical keyboard** (e.g. USB or Bluetooth) with the tablet:

| Action        | Shortcut        |
|---------------|-----------------|
| Enter digits  | `0–9`           |
| Clear PIN     | `C` on numpad   |
| Backspace     | `Backspace`     |
| Submit login  | `Enter`         |
| Go back       | `Back` key (if supported by device/OS) |

The app is mainly touch-based; most actions are done by tapping on-screen buttons.

---

## Tips

- **Server URL** – Must be correct for sync. Check Settings → Server URL (WiFi) if sync fails.
- **Printers** – Configure in Settings → Printer Setup.
- **Receipt size** – Adjust in Settings (Normal / Large / XLarge).
- **Void requests** – Require supervisor/KDS approval when enabled.
- **Closed bills** – Use Closed Bills to recall a previous bill if needed.
